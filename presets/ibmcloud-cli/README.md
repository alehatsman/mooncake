# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: ibmcloud-cli
```

## Authentication
```bash
# Login/configure
ibmcloud-cli auth login
ibmcloud-cli configure
```

## Common Operations
```bash
# List resources
ibmcloud-cli list

# Create resource
ibmcloud-cli create --name myresource

# Get status
ibmcloud-cli status

# Delete resource
ibmcloud-cli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
