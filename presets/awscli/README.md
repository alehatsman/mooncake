# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: awscli
```

## Authentication
```bash
# Login/configure
awscli auth login
awscli configure
```

## Common Operations
```bash
# List resources
awscli list

# Create resource
awscli create --name myresource

# Get status
awscli status

# Delete resource
awscli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
