# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: linode-cli
```

## Authentication
```bash
# Login/configure
linode-cli auth login
linode-cli configure
```

## Common Operations
```bash
# List resources
linode-cli list

# Create resource
linode-cli create --name myresource

# Get status
linode-cli status

# Delete resource
linode-cli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
