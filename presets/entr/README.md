# entr

Run commands when files change

## Quick Start

```yaml
- preset: entr
```

## Usage

```bash
# Basic usage
entr --help

# Common operations
entr --version
```

## Agent Use

Automation-friendly CLI tool with:
- Exit codes for error handling  
- JSON/YAML output support (where applicable)
- Scriptable interface
- Idempotent operations
