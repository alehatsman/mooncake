# vscodium

Text editor / IDE

## Quick Start

```yaml
- preset: vscodium
```

## Usage

```bash
# Basic usage
vscodium --help

# Common operations
vscodium --version
```

- Exit codes for error handling  
- JSON/YAML output support (where applicable)
- Scriptable interface
- Idempotent operations
