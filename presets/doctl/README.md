# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: doctl
```

## Authentication
```bash
# Login/configure
doctl auth login
doctl configure
```

## Common Operations
```bash
# List resources
doctl list

# Create resource
doctl create --name myresource

# Get status
doctl status

# Delete resource
doctl delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
