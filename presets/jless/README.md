# jless - JSON Viewer

Interactive JSON viewer with collapsing, search, and syntax highlighting.

## Quick Start
```yaml
- preset: jless
```

## Usage
```bash
jless data.json
curl https://api.github.com/users/github | jless
```

## Keys
- `Space` - Page down
- `/` - Search
- `q` - Quit
- `←/→` - Collapse/expand
