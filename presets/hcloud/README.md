# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: hcloud
```

## Authentication
```bash
# Login/configure
hcloud auth login
hcloud configure
```

## Common Operations
```bash
# List resources
hcloud list

# Create resource
hcloud create --name myresource

# Get status
hcloud status

# Delete resource
hcloud delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
