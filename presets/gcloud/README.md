# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: gcloud
```

## Authentication
```bash
# Login/configure
gcloud auth login
gcloud configure
```

## Common Operations
```bash
# List resources
gcloud list

# Create resource
gcloud create --name myresource

# Get status
gcloud status

# Delete resource
gcloud delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
