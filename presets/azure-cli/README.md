# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: azure-cli
```

## Authentication
```bash
# Login/configure
azure-cli auth login
azure-cli configure
```

## Common Operations
```bash
# List resources
azure-cli list

# Create resource
azure-cli create --name myresource

# Get status
azure-cli status

# Delete resource
azure-cli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
