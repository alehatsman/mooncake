# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: vultr-cli
```

## Authentication
```bash
# Login/configure
vultr-cli auth login
vultr-cli configure
```

## Common Operations
```bash
# List resources
vultr-cli list

# Create resource
vultr-cli create --name myresource

# Get status
vultr-cli status

# Delete resource
vultr-cli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
