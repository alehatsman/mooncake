# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: oci-cli
```

## Authentication
```bash
# Login/configure
oci-cli auth login
oci-cli configure
```

## Common Operations
```bash
# List resources
oci-cli list

# Create resource
oci-cli create --name myresource

# Get status
oci-cli status

# Delete resource
oci-cli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
