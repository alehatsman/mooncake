# alacritty

GPU-accelerated terminal emulator

## Quick Start

```yaml
- preset: alacritty
```

## Usage

```bash
# Basic usage
alacritty --help

# Common operations
alacritty --version
```

## Agent Use

Automation-friendly CLI tool with:
- Exit codes for error handling  
- JSON/YAML output support (where applicable)
- Scriptable interface
- Idempotent operations
