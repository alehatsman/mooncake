# nats-server

Message queue / streaming

## Quick Start

```yaml
- preset: nats-server
```

## Usage

```bash
# Basic usage
nats-server --help

# Common operations
nats-server --version
```

- Exit codes for error handling  
- JSON/YAML output support (where applicable)
- Scriptable interface
- Idempotent operations
