# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: fly
```

## Authentication
```bash
# Login/configure
fly auth login
fly configure
```

## Common Operations
```bash
# List resources
fly list

# Create resource
fly create --name myresource

# Get status
fly status

# Delete resource
fly delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
