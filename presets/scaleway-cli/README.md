# Cloud CLI Tool

Manage cloud resources from the command line.

## Quick Start
```yaml
- preset: scaleway-cli
```

## Authentication
```bash
# Login/configure
scaleway-cli auth login
scaleway-cli configure
```

## Common Operations
```bash
# List resources
scaleway-cli list

# Create resource
scaleway-cli create --name myresource

# Get status
scaleway-cli status

# Delete resource
scaleway-cli delete myresource
```

## Agent Use
- Automate cloud infrastructure
- Deploy resources programmatically
- Query cloud state
- CI/CD integration
- Infrastructure automation
